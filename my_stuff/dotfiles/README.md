# Dotfiles

this repo contains all of my dotfiles in one place.

## setup

install yadm

```sh
# Arch Linux
sudo pacman -S yadm
```

Clone dotfiles

```
# clone with https:
yadm clone https://github.com/aydenjahola/dotfiles.git

# clone with ssh
yadm clone git@github.com:aydenjahola/dotfiles.git
```
